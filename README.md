# Eye Disease identification
# 眼部疾病识别
这个是我大创的一个项目，诊断由糖尿病引发的眼部疾病。目前是用基本的CNN做的疾病分期，根据老师给的数据和Kaggle数据集合的一部分。其实可以用CAM网络做一个大概病灶位置的判断，但是自己犯懒不想做了。

## 实现
利用tensorflow+python3.5完成的，然后又用pyqt5加了一个简陋的界面。
